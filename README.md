# Neon Jump

A fast-paced, addictive endless platformer with stunning neon visuals. Built for HTML5 and optimized for CrazyGames.

![Neon Jump](screenshot.png)

## 🎮 How to Play

- **Desktop**: Press `SPACE` or click to jump
- **Mobile**: Tap to jump
- Jump on platforms to climb higher
- Collect yellow coins for bonus points (+10)
- Don't fall off the screen!
- Watch out for moving platforms (pink)

## 🚀 Features

- Smooth 60fps gameplay
- Neon glow effects and particle systems
- Progressive difficulty scaling
- High score saves locally
- Mobile-optimized touch controls
- Retro synthwave aesthetic

## 🎯 Scoring

- Gain points by climbing higher
- Collect coins for +10 points each
- Beat your high score!

## 🛠️ Technical

- Pure HTML5 Canvas (no dependencies)
- ~15KB total size
- Works offline
- Responsive design

## 📱 Platforms

- Desktop browsers
- Mobile browsers
- CrazyGames integration ready
